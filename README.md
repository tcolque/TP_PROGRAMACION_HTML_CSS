# Salta Capital

Sitio web informativo sobre la ciudad de Salta Capital, desarrollado como proyecto práctico para la materia de Programación.

## Descripción:

El objetivo de la página web es presentar información turística sobre Salta Capital mediante un sitio web moderno, organizado y responsive.

El sitio cuenta con tres páginas principales:
- Inicio: presentación general de la ciudad, imágenes, carrusel y video.
- Información: lugares turísticos, tabla informativa, acordeón y mapa.
- Contacto: formulario con validaciones utilizando HTML y JavaScript.

## Las tecnologías utilizadas son:

- HTML
- CSS
- Bootstrap 
- JavaScript
- Git y GitHub

## Cómo ejecutar el proyecto

1. Clonar el repositorio:

git clone https://github.com/tcolque/TP_PROGRAMACION_HTML_CSS.git

2. Abrir la carpeta del proyecto.

3. Ejecutar el archivo `Index.html` desde cualquier navegador.

## Funcionalidades

- Diseño responsive utilizando Bootstrap.
- Menú de navegación.
- Carrusel de imágenes.
- Tarjetas informativas.
- Tabla de información turística.
- Acordeón con información cultural.
- Mapa integrado de Google Maps.
- Formulario de contacto.
- Validación de formulario mediante HTML5.
- Validación en vivo utilizando JavaScript.
- Manipulación del DOM mediante eventos.
- Limpieza automática del formulario luego del envío.
